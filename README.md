## Installation

MPDebug is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'MPDebug'
```

### NodeJS

```shell
brew install node
cd Server
npm install
```

## Run

```shell
./run.sh
```

final go to [Server](http://localhost:3000/)
